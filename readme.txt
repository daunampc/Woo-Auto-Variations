== Description ==
Woo Auto Variations giúp bạn tạo các biến thể sản phẩm tự động theo Color và Size. Hỗ trợ pagination, bulk action, AJAX, toast notification và cập nhật tự động qua GitHub.

== Features ==
* Tự động tạo biến thể sản phẩm theo thuộc tính (Color, Size).
* Hỗ trợ phân trang (pagination).
* Thao tác hàng loạt (bulk actions).
* Cập nhật tự động thông qua GitHub.
* AJAX mượt mà + thông báo Toast UI.

== Installation ==
1. Upload plugin vào thư mục `/wp-content/plugins/`.
2. Kích hoạt plugin từ menu 'Plugins' trong WordPress.
3. Vào mục **Auto Variations** để sử dụng.
